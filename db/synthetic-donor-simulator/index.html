üéØ What This Is

A prediction tool that simulates how different donor "archetypes" would respond to SMS/MMS fundraising creatives.

Instead of A/B testing in production (expensive, slow), we predict winners before sending.

üß† Why Archetypes Work

  • Response tendencies beat demographics
  • We don't simulate "72-year-old from Ohio"
  • We simulate "person who responds to urgency cues"
  • Psychological triggers > personal details

‚úÖ Validated Results

  • SoldierStrong: 90% accuracy (9/10)
  • Patriot Donor: 10/10 correct
  • Celebrity Follower: 10/10 correct
  • Impact Seeker: 4/10 (anti-predictive)

‚öôÔ∏è How To Use

  • Paste two creatives (known winner vs loser)
  • Each archetype "votes" with reasoning
  • Majority vote = prediction
  • Adjust weights based on client data

üî¨ Key Insight

Detailed specificity is anti-predictive.

The more analytical the persona (Impact Seeker), the worse it predicts. Real donors respond to emotional triggers, not logical arguments.

üìà Improving Accuracy

  • Weight Celebrity Follower higher for nonprofit
  • Weight Urgency Responder higher for political
  • Consider dropping Impact Seeker (anti-signal)
  • Run 25+ pairs per client for statistical validity
