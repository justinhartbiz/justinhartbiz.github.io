## Key Features

### 1. DNA Extraction
Upload any SMS/MMS creative to see its complete breakdown:
- Sentence structure mapping
- Emotional triggers identification
- Persona analysis
- Word count optimization
- Weak points flagged
- Performance prediction

### 2. Campaign Comparison
Compare any two campaigns side-by-side:
- Pattern differences highlighted
- Word choice analysis
- Structural variations
- Emotional arc visualization
- Strengths/weaknesses analysis

### 3. Winner Generator
Create high-probability winning creative using:
- Client-specific formula builder
- Donor persona targeting
- Emotional trigger selection
- Seasonal optimization
- Built-in testing framework

### 4. Pattern Library
Browse the complete pattern library:
- Winning sentence structures
- Power phrases by client
- Poison words to avoid
- Emotional triggers ranked by performance
- Visual examples of successful structures

### 5. Performance Prediction
Score any creative before sending:
- Predicted Yield Index (political)
- Predicted $/send (nonprofit)
- Click-through rate estimation
- Unsubscribe risk assessment
- Improvement suggestions

## How to Use

1. **DNA Extractor:** Paste any creative into the analysis box
2. **Campaign Comparison:** Select two campaigns from dropdown or paste custom sets
3. **Winner Generator:** Select client, target persona, emotional trigger, and customize
4. **Pattern Library:** Browse the complete collection of winning patterns
5. **Performance Predictor:** Score any creative before sending
